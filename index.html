ヘルプ

        
      

      

        
前提

        
    
          
  • MetaMask をインストール済み
    • 
          
  • Soneium Minato テストネット登録済み
    • 
          
  • ETH ≥ 0.001（テストトークン）
    • 
        


        
手順


        
通常モード（MetaMask を利用する実機確認）

        
    
          
  1. 「回答を見る」でまず回答を確認します。
    2. 
          
  2. 詳細は「コンテンツの要求（支払要求）」を押すと「支払い方法の選択」ポップアップが表示されます。ここで暗号資産（ETH）を選択してください（他は無効です）。
    3. 
          
  3. 暗号資産を選択すると MetaMask の承認画面が開きます。実際のテストネットウォレットで送金を承認してください。
    4. 
          
  4. 支払いが完了すると詳細が表示されます。ページを更新すると再表示できます。
    5. 
        


        
モックモード（開発・確認用）

        
    
          
  1. モックモードは画面上部の「モックで開く」リンク、または URL に ?mock=true を付与して起動します。
    2. 
          
  2. モックモードでは実際のウォレットは不要で、ページ内のモックウォレットが支払い処理をシミュレーションします。
    3. 
          
  3. 支払いはシミュレーションのためブロックチェーン送金は発生しません。動作確認や UI 検証にご利用ください。
    4. 
          
  4. モックモード中はヘッダーに「通常モードに戻る」リンクが表示され、これで実機確認モードへ戻せます。
    5. 
        


        
補足: フッターの「状態をリセット」で初期状態に戻ります。
